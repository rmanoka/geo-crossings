pub struct Line<T> where
    T: CoordNum,  {
    pub start: Coordinate<T>,
    pub end: Coordinate<T>,
}
Expand description

A line segment made up of exactly two Coordinates.

Semantics

The interior and boundary are defined as with a LineString with the two end points.
